# BountyPool

Collaborative bounty funding platform with community contributions on Base and Stacks blockchains.

## Features

- Create bounties for tasks
- Pool contributions from community
- Track total funding
- Assign bounties to hunters
- Transparent distribution

## Tech Stack

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **Base**: Solidity 0.8.20, Foundry, Reown wallet
- **Stacks**: Clarity v4, Clarinet, @stacks/connect

## Getting Started

```bash
pnpm install
pnpm dev
```
